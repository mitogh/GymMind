#Problem H - Relational Operators

<p class="MsoNormal" align="center" style="text-align:center"><b><span style="font-size:28.0pt;font-family:Arial">Problem H</span></b><span style="font-family:Arial"><br>
</span><b><span style="font-size:20.0pt;font-family:Arial">Relational Operators</span></b><span style="font-family:Arial"><br>
<b>Input: </b>Standard Input</span></p>

<p class="MsoNormal" align="center" style="text-align:center"><b><span style="font-family:Arial">Output: </span></b><span style="font-family:Arial">Standard
Output </span></p>

<p class="MsoNormal" style="text-align:justify">&nbsp;</p>

<p class="MsoBodyText2">Some <span class="GramE">operators</span> checks about the
relationship between two values and these operators are called relational
operators. Given two numerical values your job is just to find out the
relationship between them that is (<span class="SpellE">i</span>) First one is
greater than the second (ii) First one is less than the second or (iii) First
and second one is equal.</p>

<p class="MsoBodyText2">&nbsp;</p>

<h5><span class="stylearial16pt"><span style="font-size:16.0pt">Input</span></span></h5>

<p class="MsoNormal" style="text-align:justify">First line of the input file is
an integer t (t&lt;15) which denotes how many sets of inputs are there. Each of
the next t lines contain two integers a and b (|<span class="SpellE">a|<span class="GramE">,|</span>b</span>|&lt;1000000001).<span style="mso-spacerun:yes">&nbsp;
</span></p>

<p class="MsoNormal">&nbsp;</p>

<h4><span class="stylearial16pt"><span style="font-size:16.0pt">Output </span></span></h4>

<p class="MsoNormal" style="text-align:justify">For each line of input produce
one line of output. This line contains any one of the relational operators
“&gt;”, “&lt;” or “=”, which indicates the relation that is appropriate for the
given two numbers.</p>
